Dans ce notebook, on va s'intresser l'estimation de la moyenne d'une variable alatoire donne. En effet, il s'agit d'un cas particulier d'algorithme d'approximation stochastique.

Soit $X_1,...,X_n$ une suite de variable alatoire i.i.d de mme loi. Si de plus, $\mathbb{E}(X_1)=\mu<\infty,$ la loi des grands nombres nous dit que pour $N$ assez grand, $$\overline{X}_n = \frac{1}{n} \sum_{k=1}^{n} X_k \xrightarrow[\ n \to +\infty \ ]{\ \text{p.s.}\ } \mathbb{E}(X_1).$$

Considrons la suite $\overline{X}_{n+1}$. On a $\forall n\in \mathbb{N},$

$ \begin{align} \overline{X}_{n+1} &= \frac{1}{n+1} \sum_{k=1}^{n+1} X_{k} \\ &= \frac{n}{n+1}\overline{X}_{n}+ \frac{1}{n+1}X_{n+1} \\ &=\overline{X}_n-\frac{1}{n+1}(\overline{X}_n-X_{n+1}). \end{align}$

Dans le cadre d'un algorithme d'approximation stochastique, cela correspond la suite dfinie par: $$ \begin{equation} \forall n\geq0, y_{n+1}=y_n-\gamma_{n+1}(y_n-X_{n+1}), \tag{1} \end{equation}$$ o $y_{n+1}=\overline{X}_{n+1}$, $y_n=\overline{X}_{n}$, $\gamma_{n+1}=\frac{1}{n+1}$ et $\mathbb{E}(X_1)=\mu<\infty$.

Si on considre la fonction $$ \begin{align} H:\mathbb{R}^2 &\longrightarrow\mathbb{R} \\ (y,X) &\longmapsto (y-X)^2, \end{align} $$ avec $X$ une suite de variable alatoire i.i.d de mme loi que $(X_n)_{n\geq0}$.

On a alors, $$ \begin{align} h(y) &=\mathbb{E}[H(y,X)]\\ &=\mathbb{E}[y^2-2yX+X^2]\\ &=y^2-2y\mathbb{E}[X]+\mathbb{E}[X^2]. \end{align} $$

La fonction $h$ est donc une foction dfinie de $\mathbb{R} \mapsto \mathbb{R}$. Elle est continue et drivable sur $\mathbb{R}$, car il s'agit d'une fonction polynomiale sur $\mathbb{R}$.

En tudiant la fonction $h$, on dduit qu'elle admet un unique minimum et celui-ci est $y^*=\mathbb{E}[X]$.

Approximation Stochastique¶

Nous voulons maintenant retrouver une approximation de $y^*$ par l'algorithme $(1)$.

En initialisant $y$ $0$, l'algorithme $(1)$ devient: $$ \begin{equation} \forall n\geq1, y_{n}=\sum_{k=1}^{n}(\prod_{i=k+1}^{n}(1-\gamma_i))\gamma_kX_{k}. \tag{2} \end{equation}$$

D'aprs le cours, $y_n$ converge vers $y^*$ si et seulement si $\sum_{k\geq1}\gamma_k\xrightarrow[\ n \to +\infty \ ]{\ \text{p.s.}\ }+\infty$ et $\sum_{k\geq1}(\gamma_k)^2<\infty, \text{p.s.}$

Implémentation¶

On s'intresse maintenant au cas o $X\sim\mathcal{N}(\mu,\sigma^2)$.

# 1) Paramtres de simulation
N = 10000  # Nombre total de pas de simulation
n_0 = 50  # Numro de l'tape pour mettre en avant le caractre alatoire de Y_n0
mu = 5  # Moyenne de la variable alatoire Z
sigma = 2  # cart-type de Z
gamma = lambda n: 1 / (n + 1)  # Fonction pour la taille des pas

# Initialisation des tableaux
Y = np.zeros(N + 1)  # Tableau pour stocker les valeurs de Y_n
Y[0] = 0  # Valeur initiale de Y
X_simu = np.random.normal(mu, sigma, N)  # chantillons i.i.d. de Z, tirs d'une loi normale

# Boucle d'approximation stochastique
for n in range(N):
    # Mise  jour de Y_n avec la rgle d'approximation stochastique
    Y[n + 1] = Y[n] - gamma(n) * (Y[n] - X_simu[n])

# 2) Affichage de la valeur de Y_n0 pour souligner l'effet alatoire
print(f"Valeur de Y_{n_0} (pour montrer le caractre alatoire): {Y[n_0]}")

# 3) Trac de la convergence de Y_n vers la moyenne mu
plt.figure(figsize=(14, 6))

# Graphique de convergence
plt.subplot(1, 3, 1)
plt.plot(range(N + 1), Y, label="Y_n", color="blue")  # Trac de Y_n en fonction de n

# Gnration d'chantillons i.i.d. de Xi avec une moyenne connue (mu) et une variance (sigma^2) 
mu = 5.0           # Moyenne relle de la distribution
sigma = 2.0        # cart-type de la distribution
n_samples = 10000  # Nombre d'chantillons

# Gnrer des chantillons i.i.d. d'une distribution normale avec moyenne mu et cart-type sigma
Xi = np.random.normal(loc=mu, scale=sigma, size=n_samples)

# Fonction d'estimation rcursive de la moyenne avec une squence de gain gnrale
def recursive_mean_estimator(Xi, gamma_func):
    Y = np.zeros(len(Xi))  # Tableau pour stocker les estimations rcursives
    Y[0] = Xi[0]           # Initialisation avec le premier chantillon comme point de dpart
    
    # Parcourir les chantillons et mettre  jour Y avec la formule rcursive
    for k in range(1, len(Xi)):
        gamma_k = gamma_func(k)  # Rcuprer le gain pour cette itration
        Y[k] = Y[k-1] + gamma_k * (Xi[k] - Y[k-1])  # Mise  jour rcursive
    
    return Y

# Dfinition de diffrentes fonctions de gain pour comparaison
def gamma_standard(k):
    return 1 / k   # Choix: 1/k

def gamma_sqrt(k):
    return 1 / np.sqrt(k + 1)  # Choix : 1/sqrt(k+1)

# Calcul des estimations rcursives avec les deux squences de gain
Y_standard = recursive_mean_estimator(Xi, gamma_standard)
Y_sqrt = recursive_mean_estimator(Xi, gamma_sqrt)

# Trac des rsultats pour visualiser la convergence de Y_k vers la vraie moyenne
plt.figure(figsize=(12, 6))
plt.plot(Y_standard, label=r'Gain Standard $\gamma_k = \frac{1}{k}$', color='blue')  # Courbe avec le gain standard
plt.plot(Y_sqrt, label=r'Gain Racine Carre $\gamma_k = \frac{1}{\sqrt{k+1}}$', color='green')  # Courbe avec le gain racine carre
plt.axhline(y=mu, color='red', linestyle='--', label=r'Moyenne Relle $\mu = 5$')  # Ligne pour la moyenne relle
plt.xlabel('Nombre d\'chantillons (k)')  # tiquette pour l'axe x
plt.ylabel(r'Estimation de $\mu$')  # tiquette pour l'axe y
plt.legend()  # Lgende pour identifier les courbes
plt.title(r'Estimation Rcursive de la Moyenne avec Diffrents Gains $\gamma_k$')  # Titre du graphique
plt.grid(True)  # Activer la grille pour faciliter la lecture
plt.show()  # Afficher le graphique

La figure ci-dessus nous permet de constater que le choix $\gamma_n=\frac1{\sqrt{1+n}}$ n'est pas un choix optimal.
